# moviesList-demo
CURD operation using asp.net core EF core 
--------------------------------------------------------------------------------
This project is developed using Visual Studion 2017 for Mac, Community Version
--------------------------------------------------------------------------------

How to run this project:

Step1: change connectionstring on appsettings.json
Step2: update db by using command "dotnet ef database update" on terminal
Step3: Build the project make sure all packages restored
Step4: Run the project

If you want to run in windows:
Make sure IIS settings added in Startup.cs
